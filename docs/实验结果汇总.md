# 实验结果汇总

更新日期：2026-07-21

## 评价协议

- LPW 按受试者切分，训练、验证和测试集合相互隔离。
- HMEPS 仅用于辅助训练，验证和测试仍使用 LPW。
- 模型和阈值在验证集上选择，测试集只用于最终报告。
- 主要指标为 Dice、Boundary IoU、瞳孔中心误差和空预测率。

## 软件结果

| 模型 | Dice | Boundary IoU | 中心误差 | 空预测率 |
|---|---:|---:|---:|---:|
| 128×128 LPW FP32 | `0.9021` | `0.6474` | `1.467 px` | 未统计 |
| 64×64 双线性 FP32 | `0.8549` | `0.5082` | `1.253 px` | `0%` |
| 64×64 双线性 INT8 | `0.8448` | `0.5028` | `1.415 px` | `0%` |
| 64×64 最近邻 FP32 | `0.8677` | `0.5301` | `1.319 px` | `0.6993%` |
| 64×64 最近邻 INT8 | `0.8632` | `0.5058` | `1.335 px` | `0.6993%` |

最近邻结构解决了 ESP-DL 3.3.1 对双线性 Resize 空 ROI 的解析崩溃。该结构的 INT8 Dice、Boundary IoU 和中心误差优于双线性 INT8 候选，空预测率上升至 `0.6993%`，未通过原定的零空预测门槛。

## ESP32-S3 固定图验证

| 项目 | 结果 |
|---|---:|
| 芯片 | ESP32-S3 rev0.2，16MB Flash，8MB PSRAM |
| 模型大小 | `497376` 字节（约 `485.70 KB`） |
| 模型加载后 PSRAM 占用 | 约 `619.08 KB` |
| 单次推理耗时 | `377260 us` |
| 板端阈值前景像素 | `36` |
| 电脑端阈值前景像素 | `34` |
| 板端与电脑端中心差异 | 约 `0.212 px` |

固定图闭环覆盖模型加载、输入量化、INT8 推理、阈值化和中心计算。该结果用于验证板端一致性，尚未覆盖连续推理、相机采集和完整测试集板端评价。

## OV2640 相机链路

- 已按开发板引脚表完成 OV2640 初始化和 QVGA RGB565 取帧。
- 已实现诊断用 Wi-Fi 网页预览，以及相机帧预处理、INT8 推理、最大连通区域、掩码边缘和质心叠加。
- 实机预览能够看到画面，空调遥控器发射端呈现明显强光，确认模组具备近红外响应。
- 首次取帧出现 VSYNC 溢出后，已启用 PSRAM DMA、降低 XCLK 并将相机任务调整到 CPU1；修正版构建通过，固件大小为 `0x1e8ab0`，应用分区剩余 `52%`，长期稳定性仍待定量验收。

## 当前限制

- 最近邻 INT8 候选尚未满足零空预测门槛。
- OV2640 真实眼图的掩码效果、连续 300 帧稳定性和无 Wi-Fi 端到端帧率待验证。
- 当前缺少稳定近红外补光和匹配滤光片，红外域结果仍待独立采集与标注。
- 当前没有实测能耗数据。
