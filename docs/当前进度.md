# BoundaryQAT 当前进度

更新日期：2026-07-21

## 当前结论

项目已经完成从数据准备、FP32 训练、INT8 量化到 ESP32-S3 固定图推理的工程闭环。板端模型能够正确加载和执行，输入摘要与电脑端一致，板端与电脑端预测中心相差约 `0.212 px`。

OV2640 已完成引脚接入、RGB565 取帧和 Wi-Fi 网页预览，实机确认能够响应近红外遥控信号。相机帧灰度化、缩放补边、模型推理和掩码轮廓叠加代码已经接通，但真实红外眼图效果、连续 300 帧稳定性、无 Wi-Fi 性能和红外域精度尚未验收。因此项目可以表述为“固定图板端闭环通过、相机诊断链路已接通”，不能表述为“红外端到端完成”。

## 已实现内容

- 数据切分、同步缩放补边与许可边界。
- LightUNet、训练损失、分割/边界/中心误差评价。
- ESP32-S3 约束下的 PTQ、TQT 与边界指标驱动选择。
- 双线性 Resize 失败定位、最近邻显式 ROI 修正和自动数值对齐测试。
- ESP-IDF 模型分区加载、INT8 输入量化、固定图推理、阈值与中心计算源码。
- GOOUUU ESP32-S3-CAM 引脚配置、OV2640 RGB565 取帧和 PSRAM DMA。
- 诊断用 Wi-Fi 网页预览，以及真实掩码边缘和质心叠加链路。
- 失败候选、精度门槛和未完成环节均有明确记录。

## 下一步

1. 在不回看测试集调参的前提下，决定是否坚持 `0%` 空预测率硬门槛并锁定最终模型。
2. 配置经过光生物安全评估的近红外补光和匹配滤光片，固定拍摄距离并验收真实瞳孔掩码。
3. 在无 Wi-Fi 性能模式下完成不少于 300 帧的连续采集、预处理、推理、耗时和内存稳定性验证。
4. 拍摄开发板、串口输出和实际运行过程，补充可核验的照片与视频。
5. 使用独立标注的红外样本做外部泛化验证，不以可见光结果替代红外结论。
