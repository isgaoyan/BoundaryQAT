# BoundaryQAT 当前进度

更新日期：2026-07-20

## 当前结论

项目已经完成从数据准备、FP32 训练、INT8 量化到 ESP32-S3 固定图推理的工程闭环。板端模型能够正确加载和执行，输入摘要与电脑端一致，板端与电脑端预测中心相差约 `0.212 px`。

当前尚未完成实施拍摄，也尚未完成 OV2640 连续采集、预处理、连续推理和端到端帧率验收。因此项目可以表述为“固定图板端验证通过”，不能表述为“相机端到端完成”或“实时运行”。

## 阶段进度

| 阶段 | 状态 | 关键结果 |

## 已实现内容

- 数据切分、同步缩放补边与许可边界。
- LightUNet、训练损失、分割/边界/中心误差评价。
- ESP32-S3 约束下的 PTQ、TQT 与边界指标驱动选择。
- 双线性 Resize 失败定位、最近邻显式 ROI 修正和自动数值对齐测试。
- ESP-IDF 模型分区加载、INT8 输入量化、固定图推理、阈值与中心计算源码。
- 失败候选、精度门槛和未完成环节均有明确记录。

## 下一步

1. 在不回看测试集调参的前提下，决定是否坚持 `0%` 空预测率硬门槛并锁定最终模型。
2. 接入 OV2640，完成不少于 300 帧的连续采集、预处理、推理和内存稳定性验证。
3. 拍摄开发板、串口输出和实际运行过程，补充可核验的照片与视频。
4. 获得红外采集条件后，使用独立标注样本做外部泛化验证。
