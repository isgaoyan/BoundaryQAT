# 阶段 3：边界指标驱动阈值优化结果

## 结论

> 本结果已被后续严格约束实验取代，不再作为部署候选。后续实验要求中心误差和空预测率均不得增加，最终采用 LPW+HMEPS 独立训练基线及 MinMax PTQ；详见[阶段 3：严格约束优化与辅助数据结果](阶段3_严格约束优化与辅助数据结果.md)。

阶段 3 软件侧实验已完成。固定阶段 2 的 ESP32-S3 PTQ 图和校准集后，验证集从 41 个统一阈值中选择 `0.57`。最终测试集 Boundary IoU 提升 0.0260，Dice 提升 0.0040，但中心误差增加 0.794 px、空预测率增加 0.932 个百分点。

本方法属于 `Boundary-guided tuning`，不是 QAT。阈值是统一后处理常量，不增加模型算子、输入相关分支或动态量化路径。

## 固定选择规则

- 候选阈值：0.30—0.70，步长 0.01，共 41 个。
- 标准阈值：0.50。
- 合格约束：验证 Dice 下降不超过 0.005；中心误差增加不超过 0.5 px；空预测率增加不超过 1 个百分点。
- 选择目标：合格候选中最大化 Boundary IoU；无真实收益时保留 0.50。
- 数据边界：只在验证集选择，最终测试集评价一次，不依据测试结果二次调整。

## 验证集选择

| 指标 | 标准阈值 0.50 | 选择阈值 0.57 | 变化 |
|---|---:|---:|---:|
| Dice | 0.8266 | 0.8243 | -0.0023 |
| Boundary IoU | 0.4015 | 0.4146 | +0.0131 |
| Boundary Dice | 0.5371 | 0.5455 | +0.0084 |
| 内部 Dice | 0.7940 | 0.7931 | -0.0008 |
| 中心误差 | 7.015 px | 7.415 px | +0.400 px |
| 空预测率 | 7.111% | 7.556% | +0.444 个百分点 |

阈值 0.57 满足全部预设约束，因此在读取测试集前固化为最终候选。

## 一次性测试结果

| 指标 | 标准 PTQ 0.50 | Boundary-guided 0.57 | 变化 |
|---|---:|---:|---:|
| Dice | 0.8227 | 0.8268 | +0.0040 |
| Boundary IoU | 0.4007 | 0.4267 | +0.0260 |
| Boundary Dice | 0.5352 | 0.5616 | +0.0265 |
| 内部 Dice | 0.7679 | 0.7734 | +0.0055 |
| 中心误差 | 3.041 px | 3.834 px | +0.794 px |
| 空预测率 | 2.564% | 3.497% | +0.932 个百分点 |

测试集的边界与重叠指标收益方向与验证集一致，但中心定位代价更大。该结果支持“边界指标可指导统一阈值选择”，不支持“所有任务指标同时改善”的结论。
