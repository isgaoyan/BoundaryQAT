# 阶段 4：最近邻上采样重训与量化结果

## 结论

最近邻模型的软件闭环与固定图板端推理已经完成。新 INT8 候选的 Dice、Boundary IoU 和中心误差均优于原双线性 INT8 候选，但测试集空预测率由 `0%` 增至 `0.6993%`，未满足既定“空预测率不增加”门槛，因此仍不是最终发布模型。

## 架构与导出

- 4 个解码器上采样由双线性改为最近邻；其余网络结构、训练数据、受试者切分、损失、种子与输入尺寸不变。
- 训练和普通推理使用 PyTorch 最近邻插值；ONNX 导出时显式提供全范围 ROI，避免 ESP-DL 3.3.1 的空参数解析崩溃。
- 自动测试确认 ONNX Runtime 与 PyTorch 输出一致；最终 ONNX 含 4 个 `nearest/asymmetric/floor` Resize，均无空输入。
- 最终 `.espdl` 大小 `497376` 字节，SHA256 为 `8EF3FE5A4DE0C7778572ADC842E00BA363842CA228CDD13CAB6A9E54AFB8ECEA`。

- 软件图结构、量化和精度评价已完成。
