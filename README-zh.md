# BoundaryQAT：面向 ESP32-S3 的瞳孔分割与 INT8 量化

> 当前状态：**固定图板端推理闭环已通过，实施拍摄与相机链路尚未完成。** 最近邻 INT8 候选已在 ESP32-S3 上完成模型加载、输入量化、推理、阈值化和中心计算；它仍有 `0.6993%` 测试空预测率，因此只作为工程验证候选，不作为最终发布模型。

[English](README.md) · [当前进度](docs/当前进度.md) · [数据来源与许可](docs/数据来源与许可.md)

## 项目目标

建立一条可复现的端侧瞳孔分割链路：

```text
公开眼图数据 → 轻量 UNet → FP32 训练 → ESP32-S3 兼容 INT8 量化
            → .espdl 导出 → 固定图实机推理 → 相机与实施拍摄
```

项目重点不是医疗诊断，而是在真实 MCU 约束下同时衡量 Dice、Boundary IoU、瞳孔中心误差、空预测率、内存和延迟。

## 已验证结果

| 环节 | 当前结果 |
|---|---|
| 阶段 1：128×128 FP32 | Dice `0.9021`，Boundary IoU `0.6474`，中心误差 `1.467 px` |
| 阶段 3：64×64 双线性 INT8 | Dice `0.8448`，Boundary IoU `0.5028`，中心误差 `1.415 px`，空预测率 `0%` |
| 阶段 4：64×64 最近邻 INT8 | Dice `0.8632`，Boundary IoU `0.5058`，中心误差 `1.335 px`，空预测率 `0.6993%` |
| ESP32-S3 固定图推理 | 单次 `377260 us`；板端与电脑端中心相差约 `0.212 px` |
| 板端资源 | 模型约 `485.70 KB`；加载后 PSRAM 占用约 `619.08 KB` |

板端结果来自一张锁定测试图的工程一致性验证，不能替代完整测试集精度，也不能据此宣称实时性能。

## 当前限制

- 最近邻候选虽解决了 ESP-DL 对双线性 Resize 的解析问题，但没有通过原定“空预测率保持 0%”门槛。
- 当前板端验收只覆盖固定图；连续运行、OV2640 取帧、预处理和端到端帧率尚未完成。
- 随板 OV2640 为可见光模组；红外域端到端精度仍待独立采集与标注验证。
- 实施拍摄尚未完成，仓库暂不包含演示照片或视频。


## 关键技术取舍

- 数据按受试者/完整序列隔离，避免相邻视频帧跨训练、验证和测试集合。
- INT8 使用 ESP32-S3 后端兼容的对称、二的幂、逐张量量化。
- 边界指标只指导离线配置与阈值选择，不给板端增加额外推理分支。
- 最近邻导出显式提供 ONNX Resize ROI，避免 ESP-DL 3.3.1 解析空可选输入时崩溃。
- 测试集只用于最终报告；失败候选保留限制说明，不回看测试集二次调参。
